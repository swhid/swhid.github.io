On **June 23, 2023**, the first approved version of the Software Hash Identifier (SWHID) publicly available specification is published.

This milestone concludes the first round of collaboration on the specification from the SWHID working group and provides the first stable version of the specification.

- [Version 1.0 of the specification](https://www.swhid.org/specification/v1.0/)  
- [Software Heritage announcement](https://www.softwareheritage.org/2023/06/23/swhid-publicly-available-specification/)
